# Endless Runner

A 3D endless runner game developed in Unity using C#.

## About

This project is an endless runner where the player must avoid obstacles, collect coins, and survive as long as possible while achieving a high score.

## Features

- Endless gameplay
- Coin collection system
- Score tracking
- Power-ups
- Obstacle spawning
- Character animations
- Sound effects

## Controls
- A - Strafe Left
- D - Strafe Right
- Space - Jump
## Built With

- Unity
- C#
- Visual Studio
